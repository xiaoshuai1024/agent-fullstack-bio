## 2. 组件配色原则（重要）

**在组件与页面实现中，统一使用「品牌绿」作为主色，不要使用「信息蓝」作为组件强调色。**

- **品牌绿（主色）**：`#2ECC71`（CSS 变量 `--color-primary`）。用于：主按钮、Tab 选中、筛选/标签选中、链接与高亮、搜索框聚焦环、侧栏选中、价格强调（若产品未单独规定价格色）、正向状态等。
- **不要使用信息蓝 `#3498DB`（`--color-info`）作为 UI 组件的主强调色**。`design-spec-implementation.html` 中组件演示、装饰渐变与交互态已 **统一为品牌绿**；`--color-info` 仅在 §02 色卡中保留与设计 PDF 对齐，**实现时仍映射主色**。
- **语义色保留**：警示/错误仍用 `--color-danger`；活力橙 `--color-accent` 用于次要强调、促销。**可点击组件、筛选、侧栏、链接、价格、空状态按钮** 等一律 **不用宁静蓝**，只用主色及其透明度变体。

**映射示例（Agent 实现时按此执行）：**

| 场景 | 错误做法 | 正确做法 |
|------|----------|----------|
| 筛选 Pill 选中 | 蓝色实心 | 品牌绿实心或绿描边 + 浅绿底 |
| 搜索框 focus 环 | 蓝色外发光 | 绿色外发光（`rgba(46, 204, 113, …)`） |
| 商城类目侧栏选中 | 蓝色底 | 品牌绿底或绿字 + 浅绿背景 |
| 价格数字 | 默认用蓝 | 主色绿或正文色；促销再用橙强调 |

## 3. 设计 Token（摘自 HTML `:root`）

| Token | 值 | 说明 |
|------|-----|------|
| `--color-primary` | `#2ECC71` | **品牌绿**，主交互与主强调 |
| `--color-info` | `#3498DB` | 设计体系/PDF 中的「宁静蓝」；**组件实现勿单独作强调色**，与主色同色映射 |
| `--color-accent` | `#F39C12` | 活力橙，次要强调、促销 |
| `--color-danger` | `#E74C3C` | 警示、错误、删除 |
| `--color-text` | `#2C3E50` | 主文案 |
| `--color-text-secondary` | `#7F8C8D` | 次要文案 |
| `--bg-gradient` | `linear-gradient(135deg, #F0FFF4 0%, #FFFFFF 100%)` | 页面默认铺底 |
| `--radius-lg` | `24px` | 大圆角（卡片、弹窗） |
| `--radius-md` | `16px` | 中圆角 |
| `--radius-sm` | `8px` | 小圆角 |
| `--page-padding` | `20px` | 页面水平边距 |
| `--space-component` | `16px` | 组件间距 |
| `--space-list` | `12px` | 列表项间距 |
| `--gutter` | `16px` | 栅格列间距 |
| `--nav-height` | `60px` | 导航高度（示意） |
| 字体 | Noto Sans SC + Inter | 中文标题/正文 Web 替代 |

**玻璃拟态（概念）：** 半透明白 `rgba(255,255,255,0.2~0.4)` + `backdrop-filter: blur` + 细白边；具体数值见 HTML §04、§06 及小程序相关卡片。

## 4. 字体层级（与 HTML §02 一致）

- 页面标题：Bold，约 32px（clamp 响应式）
- 模块标题：20px 量级
- 正文：16px，行高 1.5
- 辅助：14px / 12px

---

## 5. 小程序相关（与 `#sMpCommon` 对齐）

- **顶栏**：约 44px 内容区；真机需叠加状态栏与安全区 `env(safe-area-inset-top)`。
- **底栏**：五 Tab；**毛玻璃**（`backdrop-filter: blur` + 半透明白底 + 顶内高光，见 `design-spec-implementation.html` 中 `.mp-tabbar-light`）；选中色为 **`var(--color-primary)`（品牌绿）**，未选中为次要字色；底部预留 `safe-area-inset-bottom`。
- **搜索、筛选、侧栏**：按 **§2 组件配色原则**，一律用品牌绿体系，不用信息蓝作选中态。

---

## 6. 动效与无障碍

- 尊重 `prefers-reduced-motion: reduce`：减弱或关闭装饰性动画。
- 已移除整页「光感流动」类全屏扫光动画；骨架屏等轻量动画见 HTML §08。

## 7. 与原型基线文档的关系

- 共享家原型历史约定主色为 `#45aa54`（原见已删除的 `PROTOTYPE_SHARED_HOME.md`，可查 `git` 历史）；**本规范 HTML 与本文档以 `#2ECC71` 为设计稿主色**。若业务要求与潍商会共享家小程序完全一致，**以产品/设计指定为准**；Agent 默认采用本文档与 HTML 实现稿。
